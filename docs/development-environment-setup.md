# Development Environment Setup

## System Requirements

- **OS**: Windows, macOS, or Linux
- **PHP**: 8.4.15
- **Server**: LAMP, WAMP, XAMPP, or equivalent
- **SSH**: Pre-installed on most systems; install if needed

## Database

- **PostgreSQL** with pgvector extension

## Required Tools
